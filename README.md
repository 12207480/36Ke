# 36Ke
一款仿36氪iOS版APP（正在更新） <br/>
项目描述：	36氪提供创业资讯、科技新闻、投融资对接、股权投资、极速融资等创业服务,致力成为创业者可以依赖的创业服务平台,为创业者提供最好的产品和服务。 <br/>
主要功能简介： <br/>
新闻：查看科技新闻、创业资讯（已完成，并且实现了离线缓存） <br/>
发现：寻找投资人、申请融资等 <br/>
股权投资：股权投资相关新闻、募资信息等 <br/>
我的：个人用户信息、收藏等 <br/>
